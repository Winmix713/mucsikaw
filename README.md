# Button Style Playground

Interactive React playground for building and exporting modern glass-style button designs.

## Setup

```bash
npm install
npm run dev
```

## Quality scripts

- `npm run typecheck` – strict TypeScript validation
- `npm run lint` – ESLint quality checks
- `npm run lint:fix` – auto-fix lint issues
- `npm run build` – production build
- `npm run format` / `npm run format:check` – Prettier formatting

## CI quality gate

Pull requests are gated by:

1. `npm run typecheck`
2. `npm run lint`
3. `npm run build`
